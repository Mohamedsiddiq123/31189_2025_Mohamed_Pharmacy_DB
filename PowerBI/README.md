
Power BI Dashboard
Introduction

Power BI was used to create an interactive dashboard for the Pharmacy Management System. The dashboard provides a clear visualization of pharmacy data, enabling users to monitor sales performance, customer activity, medicine distribution, and supplier information. It helps managers make informed decisions through real-time data analysis.

Dashboard Components

The dashboard includes the following visualizations:

Total Sales Card: Displays the total value of all pharmacy sales.
Sales by Medicine (Column Chart): Shows the sales amount for each medicine, allowing comparison of medicine performance.
Sales by Customer (Pie Chart): Illustrates the distribution of sales among customers.
Sales by Supplier (Bar Chart): Displays sales associated with each supplier.
Filters (Slicers): Allow users to filter and analyze the data interactively.
Benefits of Using Power BI
Provides interactive and easy-to-understand reports.
Improves decision-making through visual analytics.
Helps monitor pharmacy sales and inventory performance.
Enables quick identification of top-selling medicines and active customers.
Presents data in a professional and user-friendly format.
Conclusion

The Power BI dashboard successfully transformed raw database information into meaningful visual reports. It enhanced data analysis, simplified reporting, and supported better business decisions for the Pharmacy Management System.
